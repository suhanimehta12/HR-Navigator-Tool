HR Navigator Tool

Project Overview
The HR Navigator Tool is a comprehensive AI-driven system designed to assist organizations in employee attrition prediction, promotion eligibility, and recruitment decision-making. By integrating advanced data preprocessing, machine learning models, and predictive analytics, the tool provides actionable insights for HR teams. It enables organizations to identify at-risk employees, evaluate candidates for hiring, and determine promotion readiness, thereby improving workforce management and strategic planning.

Workflow Overview

1. Load HR Data 
   Imports historical employee and candidate datasets for analysis.

2. Preprocessing
   Cleans data, handles missing values, encodes categorical variables, and prepares features for modeling.

3. Exploratory Data Analysis (EDA) 
   Analyzes employee and candidate trends, identifies patterns, and visualizes insights for decision-making.

4. Model Training and Evaluation
   Trains multiple machine learning models and evaluates performance using metrics such as accuracy, precision, recall, and F1-score.

5. Prediction and Decision Support  
   Predicts employee attrition, promotion eligibility, and candidate suitability to aid HR decision-making.

6. Visualization
   Generates interactive dashboards and visual reports to communicate actionable insights to HR teams.
